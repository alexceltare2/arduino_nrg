This library reads the Freescale MMA8652 3-axis accelerometer:
http://www.freescale.com/webapp/sps/site/prod_summary.jsp?code=MMA8652FC

This library can handle interrupts on the MMA8652. See the example for
more information.

panStamp module with on-board MMA8652 sensor:
https://github.com/panStamp/panstamp/wiki/panStamp%20NRG.-Technical%20details
